Simulation workflow

This is a clean engineering-style summary of how the FEA portion of the project was structured.

1

Geometry definition

Each loop was represented as a semicircular band using measured loop dimensions.

2

Boundary conditions

The lower ends of the semicircle were fixed while expansion displacement was applied.

3

Numerical solution

The simulation generated deformation and contour results for comparison between loop types.

4

Interpretation

Differences in loop size and width were examined through displacement behavior and visual contours.

Design takeaway

The thick and thin loop models differ geometrically, but both use approximately the same total material volume. This makes the comparison visually and mechanically interesting for FEA-focused presentation.
